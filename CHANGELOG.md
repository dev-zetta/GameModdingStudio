# Changelog

All notable changes to this project will be documented in this file.

## [0.1.4] - 2026-01-05
- **Initial public release** — first public release after ~14 years of private development.

## [0.1.4] - 2008-07-01
- **Initial public release (historical)** — original v0.1.4 (crypton-era release notes preserved).
- [FR] Added PSD support.
- [FR] Added MP3 support and full ID3 tag handling.
- Changed searching engine to a new prototype (faster and safer).
- Buffer and memory allocation changes for improved performance; many bug fixes.

## [0.1.3] - 2008-05-01
- [FR] Added ZIP, MP4, MOV, AIFF, MIDI support.
- Fixed deleting textures after conversion.

## [0.1.2] - 2008-04-26
- [FR] Added APE, M4A, JFIF (JPEG) and 7z support.
- Improved AVI searching speed; fixed zlib result handling.

## [0.1.1] - 2008-04-25
- [FR] Added PNG, APNG and ZLib support; fixed BMP support.
- Improved searching engine (rollback buffer); added new tool "UniBatch"; removed obsolete tools.

## [0.1.5] - 2008-07-14
- [FR] Added many container/format supports (SWF, EXE, DLL, 3GP, RMF, ASF (WMA/WMV), 3DS, RAF, SND, WVPK, XBG, XPR, XWMA, CAB, XNB, WAD, MDL7, 4XM, PSMF, FLI/FLC/FLX, TTA, VQF, OFR, LA, FLV, GIF (N/A), TTF (N/A), XMV (N/A), LPAC (N/A)).
- Changed chunk conversion behavior and improved MP4 searching and TGA handling for safety.

## [0.1.6] - 2008-08-01
- [FR] Added NSV, PVA, BRES, REFT, RFNA, RFNT, RSAR, RSTM, GR2, FILM, THP, DMSG, DMSC, 8SVX and more.
- Formats now include short descriptions; multiple small bugs fixed.

## [0.2.1] - 2009-04-15
- Several changes were made in this period; original log text: "A lot of changes happened, but I forgot to log them...."
